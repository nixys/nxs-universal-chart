# ML Platform

Комбинированный кейс платформы для model serving: KServe, Knative, Vault, KEDA и Prometheus monitoring.

## Используемые чарты
- `nxs-universal-chart`
- `nuc-kserve`
- `nuc-knative`
- `nuc-vault-secret-operator`
- `nuc-keda`
- `nuc-kube-prometheus-stack`

## Используемые технологии
- KServe
- Knative Serving
- Vault Secret Operator
- KEDA
- Prometheus Operator
