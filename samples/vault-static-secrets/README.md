# Vault Static Secrets

Кейс для доставки статических секретов из Vault в Kubernetes Secret и потребления их приложением через `envFrom`.

## Используемые чарты
- `nxs-universal-chart`
- `nuc-vault-secret-operator`

## Используемые технологии
- HashiCorp Vault
- Vault Static Secret
- Kubernetes Deployment
