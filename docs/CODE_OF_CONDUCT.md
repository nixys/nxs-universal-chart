# Contributor Code of Conduct

As contributors and maintainers of this project, we pledge to respect everyone who contributes by posting issues, updating documentation, submitting pull requests, providing feedback in comments, and any other activities.

Communication through project channels must be constructive and never resort to personal attacks, trolling, harassment, insults, or other unprofessional conduct.

We promise to extend courtesy and respect to everyone involved in the project, regardless of gender, gender identity, sexual orientation, disability, age, race, ethnicity, religion, or level of experience. We expect anyone contributing to this repository to do the same.

If any member of the community violates this code of conduct, the maintainers may take action, including removing issues, comments, or pull requests, or blocking accounts, as deemed appropriate.

If you are subjected to or witness unacceptable behavior, or have any other concerns, please communicate with the maintainers privately.
